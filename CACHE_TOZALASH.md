### 1-usul: Hard Refresh (Eng Tez)

### 2-usul: Browser Cache'ni To'liq Tozalash

#### Chrome / Edge:
```
1. F12 bosing (Developer Tools)
2. Network tab'ini oching
3. "Disable cache" checkboxni belgilang
4. Sahifani yangilang (F5)
```

Yoki:

```
1. Ctrl + Shift + Delete bosing
2. "Cached images and files" ni tanlang
3. Time range: "All time"
4. "Clear data" bosing
5. Sahifani yangilang
```

#### Firefox:
```
1. Ctrl + Shift + Delete
2. "Cache" ni tanlang
3. "Clear Now" bosing
4. Sahifani yangilang
```

### 3-usul: Incognito/Private Mode
```
1. Ctrl + Shift + N (Chrome/Edge)
2. Ctrl + Shift + P (Firefox)
3. Sahifani oching
```
